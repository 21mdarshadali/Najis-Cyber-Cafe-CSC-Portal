# 🏪 Najis Cyber Cafe Portal

A modern and dynamic **CSC / Cyber Cafe Web Portal** designed to manage government services, job updates, results, admit cards, and admissions from a single platform.

---

## 🔥 Features

- 🏠 Single Page Application (SPA)
- 💼 Government Job Updates
- 📊 Exam Results Management
- 🎫 Admit Card Updates
- 🎓 Admission Notifications
- 🧑‍💼 Hidden Admin Panel (Add / Edit / Delete Data)
- 📦 Sample Data Generator
- 💳 Donation System with UPI Integration
- 📱 WhatsApp Group & Channel Integration
- 🎨 Live Theme & Font Customization
- ⚡ Fast & Responsive UI (Tailwind CSS)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Tailwind CSS  
- Data SDK (for dynamic data storage)  
- Element SDK (for live configuration)

---



---

## 📸 Use Case

This project is suitable for:
- CSC Centers  
- Cyber Cafes  
- Government Service Providers  
- Local Digital Service Centers  

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in browser
3. Admin panel available via hidden trigger
